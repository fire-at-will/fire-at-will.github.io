Sort By Sort Interval

This setting lets you control how your picture is sorted. By choosing rows or columns, your picture will be sorted by rows or columns with a width of one pixel. If you choose to sort by circles, your picture will be sorted in circles, starting at the center of the image and going out from there.

ERROR: Input is not an integer!

The sort interval determines how many pixels we sort at a time. The lower this value is, the less the sorted picture will look like the original picture. Enabling random interval size gives each row or column sorted to have a different random value which maxes out at the sort interval provided. Circle sorting does not currently support sorting intervals.

About Pixel Sorting

What is Pixel Sorting?

Pixel sorting is the process of taking in pixels from a picture and rearranging them based on the numerical value of the pixels. In our case, it means that we sort pixels from darkest to lightest. What is this numerical value? Well, inside the computer, a pixel is represented by four numbers: red(R), green(G), blue(B), and alpha(A). We get the value of each pixel by simply adding the red, green, and blue numbers together!

Instructions
  1. Upload an image from your computer using the "Choose File" button.
  2. Choose your sort settings!
    1. Use the buttons under the "Sort By" tab to pick if you want to sort by rows, columns, or circles.
    2. Choose a sort interval under the "Sort Interval" tab. The higher this number is, the more the picture will be sorted.
  3. You're ready to go! Push the pink sort button and let us do the rest!
  4. Now that the sort is done, you can either sort your picture again by repeating steps 2 & 3. If you want to download the sorted picture, click the "Download" button, and if you'd like to start over, hit the "Reset" button.
How Does This Website Work Under The Hood?

The website first uses the HTML Canvas API to break down images into an array of RGBA values. The website then chooses the order in which to sort the pixels based on the settings provided, and then uses a quick sort to rearrange the pixels.

I coded this website by hand in HTML, CSS, and Javascript with a touch of JQuery every here and there. Oh, and I used Google's Material Design Lite for the fantastic theming.
